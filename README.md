# agrinho2-2026O
Agrinho 2026 é um site educativo que apresenta a importância do agronegócio sustentável, mostrando como a tecnologia e a preservação ambiental podem trabalhar juntas para garantir a produção de alimentos e a proteção da natureza. O projeto inclui informações, imagens e atividades interativas para conscientizar os visitantes sobre práticas agrícolas responsáveis.
